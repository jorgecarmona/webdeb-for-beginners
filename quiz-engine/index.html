Welcome to the Javascript Quiz!

About the Quiz

This quiz is on all topics we have seen up until now. You can do this!

Note: A few things we may have seen very little but I want you to research and learn on your ownl

  • Create code for test answers on your own local environment and use VisualStudioCode.
  • Create a new project named: "quizzes". - There will be more to come in the next weeks.
  • Create a branch named: "js-variables-primitives"
  • Create the answers for this quiz. When finished, commit your branc and create a Pull Request
  • Submit PR to your lead.

1. Create an HTML structure with a header, main content, and footer.

Comment: The header, main, and footer should be distinctly separated and span the entire width of the view.

2. Design a button with rounded corners and a shadow effect when hovered.

Comment: The button should appear raised when hovered, giving a 3D effect.

hint: See Shadow Effect

3. Create a navigation bar that has four items: Home, About, Services, Contact.

Comment: The navigation items should be in a line (horizontal) and not as a list.

4. Design a two-column layout where the left column takes up 30% of the width and the right takes up 70%.

Comment: Ensure that the columns are side by side without any gap.

5. Style a paragraph with a line height of 1.5, text color #336699, and a font size of 16px.

Comment: The text should be comfortably spaced and the color should be a medium shade of blue.

6. Design a text input field with a placeholder saying "Enter your name".

Comment: The input field should have a clear distinction from the background and show the placeholder text faintly.

7. Create a CSS animation where a circle expands its size when hovered.

Comment: The circle should smoothly increase in size when the mouse is over it.

8. Design a blockquote element with a left border color of #FF4500 and padding of 15px.

Comment: The left border should be thicker and stand out, giving emphasis to the quoted text.

9. Style an unordered list to have its bullet points appear as squares and with a color of #228B22.

Comment: The list items should have green squares as bullet points instead of the usual circles.

10. Create a responsive image that takes up 100% width on mobile and 50% width on desktop screens.

Comment: The image should adapt its size based on the screen's width.

11. Design a fixed footer that stays at the bottom of the viewport.

Comment: Even when scrolling the content, the footer should remain at the bottom of the screen.

12. Create an overlay using CSS that covers the entire screen with a background color of rgba(0,0,0,0.7).

Comment: The overlay should make the underlying content faintly visible, creating a dimming effect.

13. Design a dropdown menu that shows a list of five items when a button is clicked

Comment: The dropdown should not be visible by default but should appear below the button when it's activated.

14. Style a table with alternating row colors: #F5F5F5 for even rows and #FFFFFF for odd rows.

Comment: This should give a striped effect to the table, making it easier to read.

15. Create a tooltip that appears above an element when hovered.

Comment: The tooltip should show a brief message and be positioned directly above the element it relates to.

16. Design a progress bar that's 70% filled with a color of #4CAF50.

Comment: The progress bar should visually represent a 70% completion rate.

17. Style a heading with a text-shadow that gives it a 3D effect.

Comment: The heading should seem raised or embossed because of the shadow.

18. Create a horizontal line (hr) that has a gradient color transitioning from #FF5733 to #FFC300.

Comment: The line should blend smoothly between the two colors.

19. Design a card with a picture at the top, a title below that, and a brief description at the bottom.

Comment: All elements of the card should be well-aligned and distinctly separated.

20. Create a CSS grid layout of 3 columns and 2 rows with a gap of 20px between each cell.

Comment: The grid should neatly arrange content into the specified rows and columns with the defined gap.
